# Inception – 42 Network

## 📌 Description

**Inception** is a system administration project from the **42 Network** focused on containerization with **Docker**.  
The goal is to deploy a secure **WordPress website** using multiple Docker containers, each running a single service.

---

## 🏗️ Architecture

The mandatory part includes three services:

- **NGINX** – Web server and reverse proxy (HTTPS only)
- **WordPress (PHP-FPM)** – Application layer
- **MariaDB** – Database service
